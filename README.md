# A Unified Heat Transfer Coefficient for Phase Change Time Prediction Across Biot Number Regimes

Este repositorio contiene el código y los datos para el artículo "A Unified Heat Transfer Coefficient for Phase Change Time Prediction Across Biot Number Regimes" (Patino, 2026).

## Descripción

El artículo presenta un modelo analítico unificado para predecir el tiempo de cambio de fase en materiales con cambio de fase (PCM) para un rango extendido de números de Biot (0.01 ≤ Bi ≤ 2). El modelo se deriva de un análisis de resistencia térmica y utiliza un coeficiente global de transferencia de calor U.

## Estructura del repositorio

- `src/`: Código fuente para el modelo analítico, solución numérica y visualización.
- `data/`: Datos utilizados en el artículo.
- `figures/`: Figuras generadas por los scripts.
- `supplementary_material/`: Material suplementario con derivaciones y análisis.
- `examples/`: Ejemplos de uso.
- `tests/`: Pruebas unitarias.

## Instalación

1. Clonar el repositorio.
2. Instalar las dependencias: `pip install -r requirements.txt` (o usar `environment.yml` para conda).

## Uso

Ejecutar los scripts en `examples/` para reproducir los resultados principales.
